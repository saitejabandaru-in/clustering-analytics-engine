# 🧠 Clustering Analytics

A **scalable unsupervised learning framework** for **multi-dimensional data segmentation and pattern discovery**.

This project benchmarks multiple clustering algorithms and provides **quantitative + visual evaluation tools**, simulating real-world data science workflows used in production environments.

---

## 🧠 Overview

The **Clustering Analytics Toolkit** enables:

- Multi-algorithm clustering on high-dimensional datasets  
- Automatic validation using statistical metrics  
- Interactive visualization of clustering outputs  
- Performance benchmarking across models  

Designed to replicate how **data scientists explore unknown datasets and extract patterns at scale**.

---

## ⚙️ Core Features

### 🔬 Multi-Algorithm Benchmarking
- Side-by-side comparison of:
  - K-Means++
  - DBSCAN
  - Hierarchical Clustering
  - Spectral Clustering
  - Gaussian Mixture Models (GMM)

### 📊 Scatter Plot Visualization
- 2D projections using:
  - UMAP
  - t-SNE  
- Cluster-colored data points for intuitive pattern discovery  

### 📏 Silhouette Score Analysis
- Quantitative measurement of cluster quality  
- Helps determine optimal clustering approach  

### 📈 Algorithm Comparison Chart
- Bar chart ranking algorithms by performance  
- Clear comparison across models  

### ⚡ Real-Time Metrics
- Cluster count  
- Silhouette score  
- Data points processed  
- Iteration tracking

## 🧪 Algorithms Compared

| Algorithm     | Silhouette Score | Strengths |
|--------------|------------------|----------|
| K-Means++    | 0.82             | Fast, scalable, spherical clusters |
| DBSCAN       | 0.76             | Arbitrary shapes, noise detection |
| GMM          | 0.74             | Probabilistic, soft clustering |
| Hierarchical | 0.71             | Dendrogram-based, no k required |
| Spectral     | 0.68             | Non-convex cluster detection |
